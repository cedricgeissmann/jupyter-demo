Jupyter Notebooks¶

Das ist eine Einführung in Jupyter Notebooks, warum sie toll sind und wie man sie im Unterricht einsetzen kann.

  • Interaktives Skript
  • Einfach Verteilbar
  • Sehr viele Möglichkeiten
  • Läuft auf allen Geräten (Browser)
    • Kann aber auch lokal laufen
  • Slides können direkt erstellt werden
  • Automatisches korrigieren von Aufgaben

Erstes Beispiel¶

Slides können ganz einfach erstellt werden, und man kann auch Latex verwenden.

$$ y = m \cdot x+b$$
In [1]:
%matplotlib inline
from sympy import *
from matplotlib import pyplot as plt

x = symbols('x')
m = 2; b = -3
plot(m*x + b)

Aufgabe Schritt für Schritt vorrechnen¶

Bestimmen Sie den Punkt der auf der Gerade $f(x) = y = 2 \cdot x - 1$, der dem Ursprung am nächsten liegt.

In [2]:
# Skizze der Geraden

x = symbols('x')
f = 2 * x - 1
plot(f, xlim=[-5, 5], ylim=[-5,5])
Out[2]:
<sympy.plotting.plot.Plot at 0x7f7d82602d90>

Der Abstand $d$¶

Den Abstand zum Ursprung können wir als $d$ definieren:

$$ \begin{align*} d^2 &= x^2 + f(x)^2 \\ d^2 &= x^2 + (2 \cdot x - 1)^2 \end{align*} $$

Weil $d$ und $d^2$ das Maximum an der gleichen Stelle haben, und wir nur an der Stelle $x$ interessiert sind, können wir vereinfacht mit $d$ rechnen.

$$ d = x^2 + (2 \cdot x - 1)^2 $$
In [17]:
# d definieren und Skizzieren
d = x**2 + (2*x - 1)**2

plot(f, d, xlim=[-5, 5], ylim=[-5,5])
Out[17]:
<sympy.plotting.plot.Plot at 0x7f7d8007ce80>

Lösung¶

Der Punkt auf der Geraden $f$, der dem Ursprung am nächsten ist, ist der Punkt $P = (0.4, -0.2)$.

Vorteil von Jupyter vs andere Tools¶

Mit Jupyter können wir Code zur Verfügung stellen, der es für die SuS einfacher macht.

  • Kontrolle über Funktionalität
  • Einfach erweiterbar
  • Wenig Abhängigkeit von externen Tools
  • Könnte an Prüfungen verwendet werden

Beispiel Scheitelpunkt¶

Wir können eine Funktion zur Verfügung stellen, die den Scheitelpunkt berechnet.

In [7]:
def scheitelpunkt(a=1, b=1, c=0):
    x_s = - b/(2*a)
    y_s = (-(b**2) + 4*a*c)/(4*a)
    print(f'Der Scheitelpunkt ist ({x_s}, {y_s})')
    
scheitelpunkt(a=5, b=-4, c=1)

Der Scheitelpunkt ist (0.4, 0.2)

Algebra mit Python¶

Python hat eine Bibliothek sympy mit der symbolisches rechnen in Python möglich ist.

  • Ziemlich umfangreich
  • Kann erweitert werden
  • Workflows können vereinfacht werden
In [8]:
from sympy import *

a, x = symbols('a x')

f = a*x**2 - 2*x + 4

integrate(f, x)
Out[8]:
$\displaystyle \frac{a x^{3}}{3} - x^{2} + 4 x$

Symbolisches integrieren¶

Mit sympy können wir auch integrieren.

In [9]:
a, x = symbols('a x')
f = a * (1 - 0.25*x**2)

nullstellen = solve(f, x)
print(f'Die Nullstellen von {f} sind: {nullstellen}')

Die Nullstellen von a*(1 - 0.25*x**2) sind: [-2.00000000000000, 2.00000000000000]

F = integrate(f, (x, -2, 2))
area = 8
